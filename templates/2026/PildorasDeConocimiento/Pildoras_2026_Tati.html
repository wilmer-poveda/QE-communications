MongoDB
 ¿Que es MongoDB?

Es un sistema de gestión de bases de datos no relacionales y de código abierto, a diferencia de otros sistemas que utilizan tablas y filas, MongoDB utiliza documentos flexibles para procesar y almacenar varias formas de datos. MongoDB nos proporciona un modelo de almacenamiento de dato elástico que nos permite almacenar y consultar fácilmente tipos de datos multivariados, simplificando la gestión de base de datos para desarrolladores.
